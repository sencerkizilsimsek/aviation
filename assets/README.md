# Aircraft Images

To add custom aircraft images, save them in this folder with these names:

- `cessna172.jpg` - Cessna 172 Skyhawk
- `da40.jpg` - Diamond DA40 Diamond Star  
- `da42.jpg` - Diamond DA42 Twin Star

## Recommended Image Sources

1. **Diamond Aircraft Official**: https://www.diamondaircraft.com
2. **Cessna/Textron Aviation**: https://cessna.txtav.com
3. **AOPA**: https://www.aopa.org
4. **Wikimedia Commons**: https://commons.wikimedia.org (search for aircraft)

## Image Specifications

- Format: JPG or PNG
- Recommended size: 1280x720 or larger
- Aspect ratio: 16:9 or 4:3 preferred

After adding images, the app will automatically use them if available.
